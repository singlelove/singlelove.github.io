# singlelove.github.io
#usage

1. npm install hexo

2. npm install

3. npm install hexo-deployer-git

#update

1. use 'git add .、git commit -m "..."、git push origin hexo' to commit to hexo branch.

2. use 'hexo g -d' to publish to master branch.

#develop

1. hexo clean
2. hexo g
3. hexo s
